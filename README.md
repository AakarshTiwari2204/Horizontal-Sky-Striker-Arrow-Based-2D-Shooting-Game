# Horizontal Sky Striker: Arrow-Based 2D Shooting Game 🎮✈️

A feature-rich 2D horizontal shooting game built using **C++ and SFML**, where the player controls a fighter aircraft and battles enemies in a dynamic environment.

---

## 🚀 Features

* Smooth player movement using keyboard (WASD / Arrow Keys)
* Multiple shooting modes (Normal, Rapid Fire, Burst, Multi-shot)
* Power-up system:

  * Shield
  * Rapid Fire
  * Bullet Speed
  * Multi-shot
  * Burst Fire
* Dynamic environment with day/night cycle
* Particle effects and explosions
* Multiple enemy types (Clouds, Birds, Enemy planes)
* Level progression system
* Score tracking and lives system
* Menu and Game Over screens

---

## 🛠 Tech Stack

* C++
* SFML (Simple and Fast Multimedia Library)

---

## 🎮 Controls

* W / ↑ → Move Up
* S / ↓ → Move Down
* A / ← → Move Left
* D / → Move Right
* SPACE → Shoot
* ENTER → Start Game
* ESC → Menu / Exit
* R → Restart

---

## ⚙️ How to Run

1. Install SFML from https://www.sfml-dev.org/download.php

2. Compile:
   g++ Main.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system

3. Run:
   ./game
